# Site do GaussX

Este diretório contém a interface web oficial do **GaussX**.

A proposta do site é manter uma estrutura direta e limpa — hero central, identidade visual forte e cards de encontros — enquanto aplica animações mais intensas no carregamento, na rolagem e na interação com o cursor.

## O que este site faz

- apresenta a identidade visual do GaussX;
- organiza os encontros do projeto;
- centraliza descrições, datas e espaço para futuros materiais;
- funciona como a frente digital da iniciativa.

## Tecnologias

- **HTML**
- **CSS**
- **JavaScript puro**

## Estrutura

```text
Site_GaussX/
├── README.md
├── index.html
├── styles.css
├── script.js
└── assets/
    └── logo-gaussx.svg
```

## Onde editar

As mudanças mais frequentes ficam aqui:

- `script.js` — títulos, descrições, datas e links dos encontros;
- `styles.css` — identidade visual, motion design e comportamento dos componentes;
- `assets/` — logo e elementos gráficos.

## Publicação

O site é publicado no **Netlify** usando esta pasta como diretório de publicação:

```text
Site_GaussX
```

## Direção visual

A interface foi pensada para manter o formato que já funcionava no projeto:

- fundo escuro;
- paleta azul/roxo;
- hero central com logo e nome;
- cards grandes para os encontros;
- motion design mais forte sem trocar a estrutura principal da página.
