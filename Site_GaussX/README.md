# Site_GaussX

Arquivos do site do **GaussX**.

## Conteúdo

- `index.html`: estrutura da página
- `styles.css`: identidade visual e animações
- `script.js`: encontros, revelação na rolagem e efeitos de movimento
- `assets/logo-gaussx.svg`: logo oficial usada no site

## Observação

No Netlify, esta pasta deve ser usada como **Publish directory**.
