# Site do GaussX

Este diretório contém a interface web oficial do **GaussX**, desenvolvida para apresentar os encontros, a identidade visual e a organização acadêmica da iniciativa.

## Conceito visual

O site mantém uma estrutura simples e centralizada, inspirada na clareza do layout do Clube de Finanças, mas com motion mais forte no hero, revelação suave na rolagem e uma identidade azul/roxa mais tecnológica.

## Objetivo

A interface foi criada para:

- organizar os encontros do projeto;
- comunicar a proposta do GaussX com clareza;
- servir de base para anexar materiais, registros e conteúdos futuros.

## Tecnologias

- HTML
- CSS
- JavaScript

## Estrutura

```text
Site_GaussX/
├── README.md
├── index.html
├── styles.css
├── script.js
└── assets/
    └── logo-gaussx.svg
```

## Onde editar

- `script.js` — títulos, descrições, datas e links dos encontros
- `styles.css` — identidade visual, animações e layout
- `assets/` — logo e arquivos gráficos

## Publicação

O deploy pode ser feito no **Netlify** a partir do GitHub.
Se este projeto estiver publicado a partir de um repositório maior, a pasta publicada deve ser:

```text
Site_GaussX
```
