# GaussX

Repositório oficial do **GaussX**, clube de interesse de Física e Matemática.

Este projeto reúne o site institucional do clube, desenvolvido para apresentar os encontros, organizar materiais e fortalecer a identidade visual da iniciativa.

## Estrutura do repositório

```text
GaussX/
├── README.md
└── Site_GaussX/
    ├── index.html
    ├── styles.css
    ├── script.js
    ├── README.md
    └── assets/
        └── logo-gaussx.svg
```

## Sobre o site

O site foi criado com uma proposta visual simples, direta e elegante, mantendo:

- fundo escuro;
- paleta em azul e roxo;
- animações de entrada no hero;
- surgimento dos cards durante a rolagem;
- estrutura leve e estática, sem dependências externas de framework.

## Publicação

O deploy é feito pelo **Netlify** a partir deste repositório.

Como o site está dentro da pasta `Site_GaussX`, o campo **Publish directory** no Netlify deve estar configurado exatamente assim:

```text
Site_GaussX
```

## Atualização

Depois de alterar os arquivos e fazer `commit` + `push`, o Netlify pode publicar automaticamente a nova versão, desde que o repositório continue conectado ao projeto.

## Identidade do projeto

O **GaussX** é uma iniciativa voltada ao aprofundamento em Física e Matemática, com foco em aprendizagem colaborativa, organização dos encontros e fortalecimento da participação estudantil em atividades acadêmicas e extracurriculares.
